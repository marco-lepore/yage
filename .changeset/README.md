# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can find the
full documentation for it [in the repository](https://github.com/changesets/changesets).

## Adding a changeset

Run `npx changeset` to create a new changeset. This will prompt you to select which packages have changed
and what type of change it is (major, minor, patch). A markdown file will be created in this directory
describing the change.

When ready to release, run `npm run version-packages` to consume all changesets and bump versions.
Then `npm run release` to publish to npm.
